# Skills

딸깍팀 Claude Code 스킬 모음. 각 디렉터리가 스킬 1개 = `SKILL.md` 포함.

## 등록된 스킬

| 스킬 | 위치 | 용도 |
|---|---|---|
| **jira-creator** | [skills/jira-creator](./jira-creator/) | 딸깍 Jira 이슈 생성 (OPS · DDK) |

## 스킬 추가하는 법

1. `skills/<name>/` 디렉터리 생성
2. `SKILL.md` — Claude Code가 읽는 스킬 본문 (시스템 프롬프트처럼 동작)
3. `README.md` — 사용법·설치·주의사항 (사람이 읽는 용)
4. 본 README 표에 한 줄 추가
5. PR 생성
